# Usage
**On Mac**
```bash
eg: With Theano as backend
$ KERAS_BACKEND=tensorflow python trackgesture.py 
```
**On Windows**
```bash
eg: With Tensorflow as backend
> set "KERAS_BACKEND=tensorflow"
> python trackgesture.py 
```
